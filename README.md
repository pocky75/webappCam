# webappCam
기존의 https://github.com/pocky75/webApp 확장버전

# 1차 확장 기능
웹브라우저에서 파일을 선택해서 서버로 전송하고,
서버는 이미지 파일을 /static/uploads 폴더에 저장함
addbook.txt 파일에 내가 적은 내용 저장
